La base de datos Peliculas.csv contiene información de 3.807 películas norteamericanas que fueron calificadas en la escala de IMDB (Internet Movie Database). Además, cuenta con diferentes variables relacionadas con el año de estreno, director, actores principales, etc. El objetivo de este estudio es analizar los factores que pueden influir en que una película sea bien calificada. Cargue la base de datos y desarrolle las siguientes preguntas.

library(tidyverse)
df <- readr::read_csv("Peliculas.csv")

Pregunta 2

Como interesa estudiar la puntuación de la película, debe transformar la variable Puntuacion_IMDB, ya que esta cuenta con un formato no numérico. Cree una variable llamada Puntuacion que tome solo la puntuación en formato numérico a partir de Puntuacion_IMDB. Luego, construya el objeto prom_puntuacion que contenga la puntuación promedio para cada tipo de género predominante.

Solución:
Para hacer el cambio, es necesario primero eliminar la palabra puntos
y luego reemplazar " , " por “.”, ya que los decimales no son por coma. Finalmente, la variable al ser manipulada con gsub() debe ser transformada a número.

df <- df %>% 
  dplyr::mutate(Puntuacion = gsub("puntos", "", Puntuacion_IMDB),
                Puntuacion = gsub(" , ", ".", Puntuacion),
                Puntuacion = as.numeric(Puntuacion)
  )

Pregunta 3

Use la librería ggplot2 para construir un gráfico de la distribución de la variable Puntuacion y un gráfico que permita comparar la distribución de la puntuación por género predominante. Comente.

Solución: Un posible gráfico de distribución es el histograma. Para evaluar la distribución por categorías, una buena opción es el boxplot.

ggplot(data = df, aes(x = Puntuacion, y = ..density..)) +
  geom_histogram(bins = 30, fill = "turquoise", color = "white") +
  xlab("Género") + ylab("Densidad") +
  ggtitle("Distribución de Puntuación")

ggplot(data = df, aes(x = Genero_Predominante, y = Puntuacion, fill = Genero_Predominante)) +
  geom_boxplot(outlier.shape = NA) +
  theme(legend.position = "none") +
  coord_flip() +
  xlab("Género") + ylab("Puntuación") +
  ggtitle("Distribución de Puntuación por Género")

Podemos concluir que la puntuación tiene una leve asimetría hacia la izquierda. En cuanto a la distribución por género predominante, vemos que existen leves diferencias en la distribución de las puntuaciones, por lo tanto el género podría estar influyendo en la puntuación recibida por la película.

Pregunta 6

Cree una función que reciba el nombre del primer actor y retorne:

  • Un listado de las películas donde participó como primer actor ordenadas por puntuación más alta.
  • La cantidad de películas donde participó por género.
  • Un gráfico de la frecuencia de los géneros donde participa.

Pruebe su función con el actor Leonardo DiCaprio.

Solución:

Historial_Actor <- function(actor) {
  
  ## Filtro de base segun actor.
  Tabla <- df %>% 
    dplyr::filter(Nombre_Primer_Actor == actor)
  
  ## Rescatamos los nombres y puntuaciones de las peliculas donde participa.
  T1 <- Tabla %>% 
    dplyr::select(Nombre_Pelicula, Puntuacion) %>% 
    dplyr::arrange(desc(Puntuacion))
  
  ## Cuenta de peliculas segun genero predominante.
  T2 <- Tabla %>% 
    dplyr::group_by(Genero_Predominante) %>% 
    dplyr::count()
  
  ## Grafico de T2.
  Grafico <- ggplot(data = T2, aes(x = Genero_Predominante, y = n, 
                                   fill = Genero_Predominante)) +
    geom_bar(position = "dodge", stat = "identity") +
    coord_flip() +
    theme(legend.position = "none") +
    geom_text(aes(label = n), position = position_dodge(width = 0.9), 
              vjust = -0.25) +
    ylab("Frecuencia") + xlab("Género") +
    ggtitle(paste("Distribución de Géneros del Actor", actor))
  
  return(list(Peliculas = T1, Generos = T2, Grafico = Grafico))
  
}
